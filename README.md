# SQL Practice Repository

**Author:** Shaurya
**Repository:** SQL Practice Journey

## Overview

This repository documents my structured SQL learning journey — starting from foundational querying and moving toward advanced database programming and analytical SQL.

The focus of this repository is not only learning SQL syntax, but understanding how SQL is used for:

* Data retrieval
* Data filtering
* Data aggregation
* Relational analysis
* Business reporting
* Database automation

The repository follows a progressive learning structure where each file builds upon the previous concept.
